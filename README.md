# Looting Remnants

A Factorio mod that adds a chance to get configurable loot when destroying buildings. Perfect for PVP multiplayer games to scavenge resources from destroyed bases.

## Features
- Chance to receive loot from destroyed buildings.
- Should work with any mod (please report if it breaks something)
- UPS-friendly (all processing done at game startup)
- Multiplayer-friendly.

## Installation
### Best option: use in-game mod portal
### Manual installation
1. Download the mod from the [Releases](https://github.com/gbdor/factorio_fork_LootingRemnants/releases) page or from [Factorio mod portal](https://mods.factorio.com/mod/LootingRemnants)
2. Place the downloaded `.zip` file into your Factorio `mods` folder.
3. Start Factorio and enable the mod in the mod settings.
